Welcome to MirrorNeuron 1.0. 

For the past year, we've been working closely with engineering teams trying to move AI agents from simple scripts into production. The recurring theme we heard was **reliability**. When agents interact with APIs, databases, and the real world, things fail.

With 1.0, we are introducing a bounded sandbox execution model powered by OpenShell, alongside our resilient Redis-backed state machine.

### What's New?

- **OpenShell Sandboxing:** Execute untrusted agent code or shell commands safely without compromising your core infrastructure.
- **Infinite Execution:** Agents can pause, sleep, and resume indefinitely without holding active memory or blocking the orchestration thread.
- **Native Clustering:** Scale masterlessly across multiple nodes using libcluster.
